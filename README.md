## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies Used

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **Framer Motion** - Animation library
- **AOS** - Animate On Scroll library
- **Lucide React** - Icon library

## Sections

1. **Hero Section** - Above the fold with main headline and CTAs
2. **Who This Is For** - ICP filter showing ideal customers

4. **Solution Positioning** - Value proposition and focus areas
5. **Core Services** - Service offerings with pricing

7. **Risk Reversal** - Guarantee section
